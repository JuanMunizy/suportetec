# Suporte Técnico

Sistema de suporte ao cliente com formulário de contato e envio de e-mail via SMTP (Gmail).

## Funcionalidades

- Formulário de contato com validação
- Envio de e-mail para a equipe de suporte
- Interface moderna com animações CSS
- Prevenção de reenvio (Post/Redirect/Get)

## Como usar

```bash
php -S localhost:8000 -t .
```

Acesse `http://localhost:8000/suporte.php`
